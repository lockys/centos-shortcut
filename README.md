# Centos-record
### httpd 
- restart:
    /sbin/service httpd restart
- to start:
    /sbin/service httpd start
- to stop:
    /sbin/service httpd stop
- Where to found the httpd error log?
    /var/log/httpd/error_log

Libraries Location:  
    /usr/local/lib/python2.7.x
